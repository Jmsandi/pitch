UNDER THE HOOD

The Agent Loop

Multi-Iteration Reasoning

Unlike simple chatbots, Kai uses an iterative agent loop that can call specialized tools and refine its response up to 5 times before answering.

  • Step 1: Build prompt with conversation history
  • Step 2: Send to Geneline-X AI for reasoning
  • Step 3: If tool call detected ‚Üí Execute tool ‚Üí Loop back
  • Step 4: Final response ‚Üí Check for escalation triggers

while (iterations < this.maxIterations) {
  iterations++;

  const response = await this.callGeneline(prompt);
  const toolCall = this.toolRegistry.parseToolCall(response);

  if (!toolCall) {
    // Final response - check for escalation
    return { response, toolCallsCount, iterations };
  }
  // Execute tool and continue loop...
}
